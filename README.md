# browser_paint
browser paint app

## Getting Started

Access https://shoheihagiwara.github.io/browser_paint/ and star drawing.

## Usage

Change color by clicking color box. Or you can press number key 1 ~ 7.


### Prerequisites

None.


### Installing

Just git clone or download and access index.html.
